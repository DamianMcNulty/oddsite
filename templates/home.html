About OddBird

OddProjects

We design and build web tools for people. We built [Susy](susy.oddbird.net) for web designers because we wanted to use it, and we are core contributors to [Compass](compass-style.org), [Django](djangoproject.com) and other open-source projects.

These days we’re building the UI for [Mozilla’s](mozilla.org) new [Test Case Management System](tcm.oddsites.net) and a parcel-mapping tool for [The Providence Plan](provplan.org).

OddProcess

We use an [**iterative** process], deploying and integrating new functionality frequently and aiming to deliver usable product launches as early and often as possible.

We encourage clients to stay closely **connected** with the daily progress, offering frequent feedback and re-prioritization via [Pivotal Tracker](pivotaltracker.com), our project management tool.

In all of our projects we work exclusively with the open web stack: **accessible** semantic HTML5 markup, unobtrusive Javascript, and advanced CSS3 styling. On the server side we are a Python/Django shop, and have extensive Django experience.
